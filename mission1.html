Intervention Orange Connect Pro

Rôle : Technicien d'intervention (Alternant) | Lieu : Secteur de Flers

Contexte et Objectifs

Orange Connect Pro est une offre professionnelle complète destinée aux TPE/PME, combinant connexion internet haut débit, téléphonie IP (ToIP) et solution de secours 4G/5G. Mon rôle consiste à migrer les clients depuis leurs anciennes installations vers cette solution moderne tout en garantissant la continuité de service.

Objectifs :

  • Assurer une migration sans interruption de service prolongée
  • Configurer et tester l'ensemble des équipements (Livebox 7 Pro, Airbox, postes DECT)
  • Former le client à l'utilisation des nouveaux équipements

Tâches Réalisées

1. Phase de Diagnostic et Préparation

Avant toute intervention, je réalise un audit complet pour garantir une migration sans perte de service :

  • Vérification SVI : Test de l'accueil téléphonique et des messages via la plateforme ATS Studio
  • Sauvegarde configuration : Récupération des paramètres réseau de l'ancienne box (DHCP, redirections de ports, Wi-Fi)
  • Validation client : Information sur les étapes et obtention de l'accord formel avant coupure

2. Déploiement Matériel et Configuration

  • Installation physique : Remplacement de l'ancienne box, raccordement de la Livebox 7 Pro et de l'Airbox de secours 4G/5G
  • Configuration réseau : Restauration des paramètres sauvegardés, activation du mode professionnel via l'interface Web
  • Téléphonie IP : Paramétrage de la ligne SIP, installation du poste fixe et appairage DECT des combinés sans fil
  • Tests fonctionnels : Vérification des appels entrants/sortants, du SVI, de la connexion internet et du basculement sur l'Airbox
  • Clôture administrative : Validation de l'intervention via l'application MOBI

3. Gestion d'Incidents Techniques

Durant mes interventions, j'ai été confronté à des anomalies nécessitant une analyse approfondie :

  • Forçage de profil Pro : Sur certaines box mal provisionnées, le mode professionnel n'était pas activé automatiquement. J'ai dû contacter le service Connect Pro pour forcer le profil à distance
  • Erreur d'appairage DECT : Échec de l'association entre la borne et les combinés causé par une inversion des adresses MAC dans le système. Résolution par correction de la base de données avec le support technique

Analyse Réflexive

Cette mission m'a permis de développer une plus grande autonomie technique et relationnelle. Sur le plan technique, j'ai appris à diagnostiquer rapidement les dysfonctionnements (erreurs de provisionnement, problèmes MAC) et à utiliser efficacement les outils métier (MOBI, ATS Studio, interfaces d'administration).

Sur le plan relationnel, j'ai amélioré ma capacité à communiquer avec différents interlocuteurs : rassurer le client pendant les coupures, expliquer les procédures de manière pédagogique, et collaborer efficacement avec les supports techniques distants.

Points d'amélioration identifiés : Mieux anticiper les temps d'intervention en intégrant systématiquement une marge pour les incidents imprévus, et renforcer ma documentation des configurations pour faciliter les interventions futures.
